# Driven3 Recovery Website

A modern, responsive single-page website for [Driven3 Recovery](https://www.instagram.com/driven3recovery/) — 24/7 breakdown recovery and vehicle transport in Oldham, Greater Manchester.

## Preview locally

Open `index.html` directly in your browser, or run a simple local server:

```bash
npx serve .
```

## Before going live

Replace placeholder contact details in `index.html`:

- **Phone:** Search for `tel:+440000000000` and `+44 0000 000000` — update with your real number (e.g. `tel:+447809191292`)
- **Email:** Update `info@driven3recovery.co.uk` if you use a different address

## Deploy

This is a static site — deploy to any static host:

- Netlify (drag & drop the folder)
- GitHub Pages
- Vercel
- Any web hosting cPanel

## Structure

```
index.html          Main page
css/styles.css      Styles
js/main.js          Navigation, scroll effects, form
assets/images/      Logo (Instagram) + stock photos (Unsplash)
```

Gallery and hero images are from [Unsplash](https://unsplash.com) (free to use). Logo is from the Driven3 Recovery Instagram profile.
